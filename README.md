# MeowTiers Legal

This repository contains the legal documents for the MeowTiers Discord bot.

MeowTiers is the unofficial ocetiers bot that allows users to quickly view OCETiers player tiers and profiles.

## Files

terms.md  
Terms of Service for the MeowTiers bot.

privacy.md  
Privacy Policy explaining what data the bot may process.

## About MeowTiers

MeowTiers uses the OCETiers API to provide commands such as:

- Viewing player profiles
- Checking tiers for specific gamemodes
- Finding a player's best tier

The bot is not affiliated with OCETiers and is provided as a community utility.
